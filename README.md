# 🎬 Netflix Clone

🔗 Acesse o Projeto:
https://arielyllopes.github.io/Netflix_clone/

Projeto desenvolvido com HTML, CSS e JavaScript inspirado na interface da Netflix, com o objetivo de praticar conceitos de desenvolvimento web, estruturação de páginas, estilização e organização de componentes.

## 📖 Sobre o Projeto

Este projeto reproduz a experiência visual da plataforma Netflix, contendo páginas de navegação, catálogo de filmes e perfis de usuário.

O objetivo principal foi aprimorar conhecimentos em desenvolvimento front-end, trabalhando com:

- Estruturação semântica em HTML5
- Estilização com CSS3
- Organização de arquivos e componentes
- Manipulação de elementos utilizando JavaScript
- Responsividade e experiência do usuário

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- Git e GitHub
